# Employee-Break-Productivity-Analysis-Dashboard
Analyzed employee activity data to track different types of breaks such as lunch, normal breaks, meetings, training, and other work activities. Calculated total break time and evaluated productivity patterns using Excel. This project involved data cleaning, time calculations, and reporting to understand employee work behavior and efficiency. 
1️⃣ Project Objective
The objective of this project is to analyze employee productivity, working hours, and break patterns using Excel. The dashboard helps management monitor employee efficiency, identify productivity trends, and track time spent on various activities such as lunch, meetings, training, and breaks.
2️⃣ Dataset Overview
The dataset contains operational data of employees including:

Employee Name & ID

Lunch Break Time

Normal Break Time

Feedback Time

Training Time

Meeting Time

Unwanted Break Time

Other Work Activities

Total Break Time

This data helps evaluate how employees spend their working hours.

3️⃣ Data Cleaning & Preparation 
The first step involved preparing the raw dataset for analysis.

Steps performed:

Organized employee records in tabular format

Checked and corrected time formats

Removed blank or inconsistent values

Standardized activity categories

Converted time data into analyzable format

This ensured accurate calculations for further analysis.
4️⃣ Break Time Analysis

In the Break sheet, employee break activities were analyzed.

Metrics calculated:

Lunch break duration

Normal break duration

Training and meeting time

Unwanted break time

Other work activities

Total break time per employee

This analysis helps identify excessive break patterns and time utilization.
Work Efficiency Calculation

In the Efficiency sheet, employee productivity was calculated using key performance indicators.

Metrics included:

Total Working Hours

5️⃣ Work Efficiency Time

Number of Calls Attended

Efficiency Percentage

Active Time with Customers

Overall Active Time

Formulas were used to calculate productivity ratios and efficiency percentages.
6️⃣ Performance Metrics Creation

Key KPIs were created to measure employee performance:

Employee efficiency percentage

Call handling activity

Active working time

Non-productive time

Overall productivity levels

These metrics help compare performance across employees.
7️⃣ Summary Analysis

A summary sheet was created to consolidate key insights such as:

Average employee efficiency

Total working hours

Break time distribution

Productivity trends

This allows management to quickly understand overall performance.
8️⃣ Dashboard Creation

An interactive Excel dashboard was developed to visualize the insights.

The dashboard includes:

Employee performance charts

Efficiency comparison graphs

Break time analysis

Productivity indicators

Key performance metrics

Visual elements such as charts, conditional formatting, and KPIs were used for better interpretation.
